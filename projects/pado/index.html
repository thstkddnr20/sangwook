파도

AI 채팅 응답 속도 개선과 모니터링을 도입하여 안정성을 확보한 프로젝트

2025.12 ~ 2026.02 (3개월)
2026.02 ~ 운영중

프로젝트 개요

일상 속 불안을 느끼지만 전문적인 도움을 받기 어려운 사람들을 위해,
ACT(수용 전념 치료) 기반의 콘텐츠와 AI 채팅을 통해
스스로 마음을 돌볼 수 있는 환경을 제공합니다.

기술 스택

Java Spring Boot JPA Docker PostgreSQL Redis RabbitMQ Prometheus Grafana

기술적 고민

채팅 데이터 유실 방지를 위한 메시지 큐 도입

Redis List를 캐시가 아닌 저장소로 사용하면서 발생한 데이터 유실 가능성을 인식,
RabbitMQ를 도입해 at-least-once 보장과 Dead Queue 기반 복구 가능 구조 구성

외부 API 호출 안정성 확보를 위한 제어·탐지·분석 체계 구축

JSON 구조화 로그, 커스텀 메트릭, 이메일 알림을 연동해
장애를 빠르게 탐지할 수 있는 모니터링 환경 구축
